# 修仙 AI 模拟器

一个以 AI 驱动的纯文字修仙模拟游戏。

## 功能特性

- 完整的角色属性系统（血量、灵力、神识、境界、灵根）
- 物品系统（法宝、丹药、灵石）
- 记忆系统（记录角色的历史经历和目标）
- AI 交互系统（提示词构建、AI 响应解析、属性更新）
- 角色探测和交互系统（多角色同步场景）
- 二维地图系统（角色坐标、范围约束、可视化地图面板）
- 境界突破系统（小境界和大境界）
- 战斗伤害计算系统
- 数据管理系统（游戏状态存数据库，法宝/功法模板存 YAML 配置文件）

## 技术栈

- Python 3.13+
- pygame (UI 框架)
- uv (包管理器)
- SQLite (运行时数据存储，如游戏状态/记忆)
- YAML (世界配置、法宝模板、功法模板)

## 安装和运行

### 使用 uv 安装依赖

```bash
uv sync
```

### 运行游戏

```bash
uv run python src/main.py
```

## 项目结构

```
ai-sim/
├── src/
│   ├── character/      # 角色系统
│   ├── item/           # 物品系统
│   ├── ai/             # AI 交互
│   ├── interaction/    # 战斗、探测系统
│   ├── realm/          # 境界系统
│   ├── world/          # 世界管理
│   ├── game/           # 游戏主循环
│   ├── ui/             # UI 渲染
│   └── main.py         # 主入口
├── config.yaml         # 基础配置文件
├── config/
│   ├── treasures.yaml   # 法宝模板配置
│   └── techniques.yaml  # 功法模板配置
└── pyproject.toml      # 项目配置
```

## 操作说明

- 按 `ESC` 或关闭窗口退出游戏
- 左下角地图面板会显示角色实时坐标与相对位置
- 地图边界由 `config.yaml` 中 `map` 配置控制
